# Treats Images

This folder contains images for the "Build your spread" section on the homepage.

## Required Images

Please add the following images:

1. **pumpkin-fritters.png** - Image of pumpkin fritters (sweet finish category)
2. **bread-options.png** - Image of loaf and vetkoek (bread bar category)
3. **braai-extras.png** - Image of braai items like samoosas, boerewors, etc. (feast mode category)

## Image Specifications

- Format: PNG or JPG
- Recommended size: 800x600px or similar aspect ratio
- Will be displayed at 264px height on the homepage
